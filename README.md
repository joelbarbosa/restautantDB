# restautantDB
Just a code challenge with Angular and Node

# Project
The project are split in two projects:

```bash

1 - Frontend (front-api)
2 - Backend (server-api)

```

## Installation BackEnd
```bash

$ cd server-api
$ npm install 

```

## Tests BackEnd
```bash

$ cd server-api
$ npm test 

```

## Run BackEnd
```bash

$ cd server-api
$ npm start 

```

## Installation FrontEnd
```bash

$ cd front-api
$ npm install

```

## Tests FrontEnd
```bash

$ cd front-api
$ npm test 

```

## Run FrontEnd
```bash

$ cd front-api
$ npm start 

```
